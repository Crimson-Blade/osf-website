# Official Website

## About

This is the official website for the Manipal OSF organisation.

## Contributing

A prerequisite knowledge of React, Next.js, TailwindCSS and basic HTML is required for the frontend and Strapi for the backend along with docker, GitHub Actions for CI/CD.

Although if you are not familiar with all of these, you can still contibute in a particular domain or by finding bugs in the site.
